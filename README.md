CRUDS Application

A dynamic CRUDS (Create, Read, Update, Delete, Search) web application
built with HTML, CSS, and JavaScript.

## ✨ Features
- ➕ Create new products with (title, price, taxes, ads, discount, count, category)
- 📋 Read and display all products in a structured table
- ✏️ Update existing product data
- 🗑️ Delete products
- 🔍 Search products by title or category
- 💰 Auto-calculate total price
- 🎨 Interactive UI with hover effects on buttons

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
